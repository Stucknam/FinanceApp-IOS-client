# FinanceApp — клиент-серверное iOS‑приложение для учёта финансов

FinanceApp — учебное мобильное приложение, разработанное в рамках курсовой работы по дисциплине «iOS‑разработка».  
Проект демонстрирует применение архитектурного паттерна MVVM, работу с REST API, разделение слоёв и модульную архитектуру.

---

## Цели проекта

- Изучение современных архитектурных подходов в мобильной разработке.
- Реализация клиент‑серверного приложения на основе MVVM.
- Исследование особенностей взаимодействия ViewModel и сетевого слоя.
- Демонстрация обработки асинхронных данных и управления состоянием.

---

## Архитектура проекта

Приложение построено на основе паттерна MVVM с чётким разделением ответственности.

### Основные слои:

- **Models** — доменные модели приложения.
- **DTO** — структуры данных, получаемые из API.
- **Mappers** — преобразование DTO в доменные модели.
- **Services** — сетевой слой, работа с REST API.
- **ViewModels** — бизнес‑логика, загрузка данных, управление состояниями.
- **Views** — интерфейс на SwiftUI.
- **Components** — переиспользуемые UI‑элементы.
- **Extensions** — вспомогательные расширения.

Такое разделение обеспечивает модульность, тестируемость и удобство сопровождения.

## Клиент‑серверное взаимодействие

Приложение использует REST API для:

- получения списка транзакций  
- получения категорий  
- получения счетов  
- создания новых транзакций  

Сетевой слой реализован через:

- URLSession  
- async/await  
- универсальный NetworkService  
- обработку ошибок и статусов ответа  

---

## Функциональность

- Просмотр баланса и диаграммы расходов/доходов.
- Просмотр категорий с процентным распределением.
- Просмотр списка транзакций.
- Добавление новой транзакции.
- Загрузка данных с сервера.
- Обработка состояний:
  - загрузка
  - ошибка
  - пустой список

## Запуск проекта

1. Клонировать репозиторий:
   ```sh
   git clone https://github.com/Stucknam/FinanceApp-IOS-client.git  
2.Открыть проект в Xcode:  
  ```sh
   open FinanceApp.xcodeproj
  ```
3.Убедиться, что API доступно.  
Приложение использует внешний REST API, расположенный в отдельном репозитории:
```sh
https://github.com/Stucknam/FinanceApp
```
Перед запуском мобильного приложения необходимо развернуть API согласно инструкции в соответствующем репозитории.

4.Запустить на симуляторе или устройстве.
